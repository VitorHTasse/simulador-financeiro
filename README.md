# 💰 Simulador de Aplicações Financeiras

Aplicação web desenvolvida com foco em educação financeira e simulações de investimentos, permitindo que o usuário visualize projeções futuras com base em diferentes cenários.

## 📌 Sobre o Projeto

O sistema permite realizar diversos cálculos financeiros essenciais para planejamento pessoal e tomada de decisões estratégicas.

## 🚀 Funcionalidades

- 📈 Calculadora de Juros Compostos
- 📊 Calculadora de Juros Simples
- 👴 Simulador de Aposentadoria
- 💼 Simulador de Investimentos
- 🏆 Simulador do Primeiro Milhão
- 💳 Simulador de Amortização de Dívidas
- ⏳ Quanto Custa Seu Tempo?
- 🎯 Atingindo Metas Financeiras

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## 🎯 Objetivo

Aplicar conceitos matemáticos e financeiros em um sistema interativo, reforçando conhecimentos em lógica de programação e manipulação de dados.
